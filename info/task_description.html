We have some statistics about horse races (time for each horse in each race).
You have to find the number of the horse which has the most wins.
For example: if the results are [[“1:13”, “1:26”, “1:11”], [“1:10”, “1:18”, “1:14”], [“1:20”, “1:23”, “1:15”]] than the 3rd horse is the fastest, because she has 2 wins at 3 races.
Every element in the list - is a string in format m:ss, for example "1:05" or "2:22". 1:00 <= time <= 5:00

Input: Racing time as array of arrays.

Output: Number of the fastest horse, which has the most wins (Important: in this task the numbers of horses are started from "1", not from "0")

Example:

{% if interpreter.slug == "js-node" %} 
fastestHorse([[“1:13”, “1:26”, “1:11”], [“1:10”, “1:18”, “1:14”], [“1:20”, “1:23”, “1:15”]]) == 3
{% else %} 
fastest_horse([[“1:13”, “1:26”, “1:11”], [“1:10”, “1:18”, “1:14”], [“1:20”, “1:23”, “1:15”]]) == 3
{% endif %}

How it is used: For data analysis and statistics (finding elements which we want in the huge amount of data).

Precondition:
2 <= horses <= 10
1 <= races <= 10
1:00 <= time <= 5:00
There is only 1 fastest horse in each test
