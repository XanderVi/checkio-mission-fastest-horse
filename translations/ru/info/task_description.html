У нас есть некоторые статистически данные о скачках (время для каждой лошади в каждой гонке).
Ваша задача - найти самую быструю лошадь. Ту, которая в исследуемой выборке имеет наибольшее количество побед.
Например: если результаты забегов выглядят следующим образом [[“1:13”, “1:26”, “1:11”], [“1:10”, “1:18”, “1:14”], [“1:20”, “1:23”, “1:15”]] Тогда самой быстрой лошадью будет №3, так как она победила в 2 забегах из 3.
Время будет задано в формате м:сс (например, "1:18" или "2:22") и находиться с пределах 1:00 <= время <= 5:00. Также обратите внимание, что каждый элемент списка имеет формат string.

Входные данные: Время, которое показали лошади в каждом забеге.

Выходные данные: Номер самой быстрой лошади. Той, которая одержала наибольшее количество побед (Примечание: в этой задаче номера лошадей начинаются с "1", а не с "0")

Пример:

{% if interpreter.slug == "js-node" %} 
fastestHorse([[“1:13”, “1:26”, “1:11”], [“1:10”, “1:18”, “1:14”], [“1:20”, “1:23”, “1:15”]]) == 3
{% else %} 
fastest_horse([[“1:13”, “1:26”, “1:11”], [“1:10”, “1:18”, “1:14”], [“1:20”, “1:23”, “1:15”]]) == 3
{% endif %}

Как это используется: Для анализа информации и статистической оценки, для поиска нужного элемента в огромном массиве данных.

Предусловия:
2 <= количество лошадей <= 10
1 <= количество забегов <= 10
1:00 <= время <= 5:00
В каждом из тестов будет только 1 самая быстрая лошадь
